> A complete, offline-capable personal finance management system built with Python Flask and JSON persistence. Perfect for individuals and families who want to take control of their finances.

## 🌟 Features

### 💰 Core Features
- **Multi-User Support** - Create and manage multiple user accounts
- **Wallet Management** - Manage multiple wallets (Physical Cash, Mobile Money, Bank Accounts)
- **Transaction Management** - Add, Edit, and Delete transactions easily
- **Investment Tracking** - Track all your investments in one place
- **Smart Alerts** - Receive intelligent financial insights and warnings
- **Dark/Light Theme** - Toggle between dark and light themes

### 📊 Financial Management
- **Income Tracking** - Record all your income sources
- **Expense Tracking** - Categorize and track every expense
- **Budget Reports** - Detailed financial reports and analytics
- **Monthly Evolution** - Track your financial evolution over time
- **Category Management** - Create custom expense/income categories

### 🔄 Advanced Features
- **Wallet Transfers** - Transfer money between wallets (Deposit/Withdraw)
- **Balance Verification** - Smart discrepancy detection
- **Financial Insights** - Get personalized financial recommendations
- **Export Data** - Export all your financial data (JSON format)
- **Multi-Currency** - Support for multiple African and World currencies
